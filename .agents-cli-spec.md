# Agent Spec

## Overview
`netsentinel-coordinator` acts as the root orchestrator (Site Reliability Engineer) for an infrastructure monitoring platform. It coordinates tasks related to monitoring, diagnosing, and resolving infrastructure issues.

## Example Use Cases
- Triage infrastructure alerts
- Delegate specialized diagnostics
- Formulate remediation plans
